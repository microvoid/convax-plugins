# Hello Convax

Use this Skill only when the user asks to verify or understand the Hello Convax
Plugin.

1. Confirm the active Canvas contains a `hello-convax` Plugin node.
2. Ask the user to use **Refresh context** if the surface still says it is waiting.
3. Explain that the displayed Project, Canvas, and node identity is host-scoped and
   cannot be selected by Plugin arguments.
4. Do not edit `.convax` files or claim that this Skill grants Plugin permissions.
